# SC-013 Нейротренировки

## Суть

Мобильный топовый сценарий каталога для продвинутых участников, которым нужен не базовый ликбез, а практическое применение нейроинструментов.

## Сценарный план

1. Быстрая проверка уровня группы
2. Практика на одном-двух нейроинструментах
3. Демонстрация результатов
4. Ссылка на запись следующего потока
