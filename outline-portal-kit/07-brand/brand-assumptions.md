# Бренд-система по брендбуку МД

Основание: [Брендбук МД.pdf](/Users/johndoe/Documents/МД/Документы/Брендбук%20МД.pdf)

## Визуальное направление

- дружелюбный городской сервис;
- чистая типографика;
- сочетание ярких заметных и "дорогих" оттенков;
- светлый фон;
- высокий контраст и крупная навигация для аудитории 55+.

## Основные цвета

- изумрудный: `RGB 22 88 73` / `#165849`
  CMYK `87 42 71 35`
- бирюзовый: `RGB 46 179 152` / `#2EB398`
  CMYK `74 4 51 0`
- рубиновый: `RGB 128 0 54` / `#800036`
  CMYK `33 100 63 34`
- красный: `RGB 189 83 109` / `#BD536D`
  CMYK `23 81 42 3`
- белый: `RGB 255 255 255` / `#FFFFFF`

## Дополнительные цвета

- темный бирюзовый: `RGB 32 133 114` / `#208572`
- светлый бирюзовый: `RGB 113 200 186` / `#71C8BA`
- очень светлый бирюзовый: `RGB 200 231 222` / `#C8E7DE`
- темный малиновый: `RGB 148 44 81` / `#942C51`
- светлый розовый: `RGB 224 137 158` / `#E0899E`
- пудровый розовый: `RGB 243 202 213` / `#F3CAD5`

## Типографика

- основной шрифт проекта: `Circe`
- разрешенные начертания:
  `Thin`, `Light`, `Regular`, `Bold`, `Extra Bold`
- для заголовков и афиш использовать `Circe Bold` или `Circe Extra Bold`
- для интерфейсного текста и описаний использовать `Circe Regular` или `Circe Light`

## Фирменное написание

- допустимо использовать фирменное написание в формате логотипа;
- допустимы любые цветовые сочетания из основной палитры;
- приоритетное написание `Московское долголетие` — фирменным бордовым цветом;
- не рекомендуется дополнительно ставить фирменное написание в формате логотипа, если на макете уже есть крупное акцентное написание названия.

## Принципы UI и афиш

- не использовать мелкий текст на афишах;
- заголовок максимум в 2 строки;
- дата и время всегда в первом экране;
- у карточки события должна быть одна главная польза;
- избегать визуального шума и перегруженных фонов;
- использовать крупные цветовые плоскости и чистую композицию;
- яркие бирюзовый и красный подходят для заметных акцентов;
- рубиновый и изумрудный подходят для более солидного, статусного визуального тона.

## Практические правила для портала

- hero-блоки страниц строить на белом фоне с бордовым или изумрудным акцентом;
- карточки событий делать светлыми, с одной цветной меткой формата;
- CTA-кнопки: бирюзовые как основные, рубиновые как акцентные;
- афиши `1080x1920` собирать на белом или чистом цветном фоне без текстур;
- логотип и крупное название не должны спорить между собой на одной афише.
