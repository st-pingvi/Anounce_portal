# Структура Outline

## Workspace

Название workspace:

`Московское долголетие — Портал анонсов`

## Коллекции

### 1. Портал анонсов

Публичная коллекция. Содержит:

- главную страницу;
- обзор "Что нового";
- страницу "Как записаться";
- страницу "Подобрать событие";
- страницу "Контакты и площадки".

### 2. Каталог событий

Публичная коллекция. В ней живут карточки мероприятий по вложенной структуре:

- `2026`
- `2026 / Апрель`
- `2026 / Май`
- `2026 / Июнь`

Внутри месяца:

- документ-витрина месяца;
- вложенные документы событий.

### 3. Афиша месяца

Публичная коллекция. Документы:

- `Апрель 2026`
- `Май 2026`
- `Июнь 2026`

Каждый документ содержит:

- hero-блок;
- подборки;
- врезку календаря;
- embed афиш;
- ссылки на карточки событий.

### 4. Календарь

Публичная коллекция. Документы:

- `Общий календарь`
- `Здоровье`
- `Творчество`
- `Цифровая грамотность`
- `По районам`

Каждый документ строится вокруг Google Calendar embed и быстрых ссылок.

### 5. Редакция

Внутренняя коллекция:

- регламент публикации;
- правила нейминга;
- статусные представления Airtable;
- инструкция по Canva и Zapier;
- чек-листы выпуска.

### 6. Шаблоны

Внутренняя коллекция:

- шаблон карточки события;
- шаблон афиши месяца;
- шаблон спецподборки;
- шаблон страницы площадки;
- шаблон редакционного дайджеста.

## Главная навигация

Порядок ссылок в левой панели и на главной:

1. `Сегодня`
2. `Афиша недели`
3. `Афиша месяца`
4. `Каталог событий`
5. `Календарь`
6. `Площадки`
7. `Записаться`

## Рекомендованные иконки/цвета коллекций

- `Портал анонсов` — orange / icon `sparkles`
- `Каталог событий` — teal / icon `calendar`
- `Афиша месяца` — pink / icon `image`
- `Календарь` — blue / icon `clock`
- `Редакция` — gray / icon `settings`
- `Шаблоны` — purple / icon `shapes`

## Шаблоны Outline

Согласно официальной документации Outline, шаблоны создаются через `Templatize`, поддерживают placeholders и переменные `{date}`, `{datetime}`, `{author}`. В данном проекте нужно создать workspace templates для:

- `Карточка события`;
- `Афиша месяца`;
- `Страница площадки`;
- `Спецподборка`;
- `Редакторская публикация`.

## Embeds в документах

Outline поддерживает embeds Airtable, Typeform, Google Suite и произвольный `/embed`.

В этом портале используются:

- Airtable view embed;
- Airtable form embed или Tally form embed;
- Google Calendar public embed;
- Canva public design link;
- Google Maps embed для схемы проезда.

## Права доступа

- Public reading включен только для публичных коллекций.
- В `Редакции` и `Шаблонах` доступ только у редакторов и администраторов.
- Public sharing для `Шаблонов` и `Редакции` отключен.
- Создание новых коллекций обычными участниками отключено.

## Стандарты именования документов

- карточка события: `YYYY-MM-DD — Название события`
- афиша месяца: `Апрель 2026`
- площадка: `ЦМД <название площадки>`
- подборка: `Подборка — <тема>`
