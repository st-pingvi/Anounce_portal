# Prototype UI

Статический визуальный прототип интерфейса репозитория.

## Что внутри

- `index.html` — экран каталога сценариев.
- `styles.css` — визуальная система и адаптивная сетка.
- `app.js` — демо-данные по 10 пилотным сценариям и логика фильтрации.

## Как открыть

Откройте файл `index.html` в браузере или поднимите любой простой статический сервер в папке `prototype-ui`.
