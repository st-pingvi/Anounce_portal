# Конституция Контента ЦМД

Документ фиксирует методические правила качества сценариев для сети центров.

## Чек-лист проверки сценария

Каждый сценарий рекомендуется проверять по 10 вопросам.

1. Есть ли реальный запрос от участников.
2. Соответствует ли формат месту проведения.
3. Учитывает ли тема уровень развития аудитории.
4. Доступно ли мероприятие для всех желающих.
5. Борется ли проект за время участника с конкурентными активностями календаря.
6. Соблюдены ли авторские права.
7. Безопасна ли тема для психологического климата.
8. Соответствует ли площадь помещения количеству участников.
9. Есть ли ресурсы у центра на проведение.
10. Понятны ли каналы привлечения участников.

## Красные флаги

Если обнаружен хотя бы один красный флаг, сценарий требует переработки или блокировки.

- Эйджизм и стереотипы
- Бесполезный формат
- Политика и религия
- Исключение участников
- Прямая реклама

## Системное предупреждение

При нарушении Конституции система должна сообщать о трех типах риска:

- перегрузка ресурсов;
- токсичная среда;
- низкая ценность мероприятия.

Рекомендуемое действие: проверить физические возможности центра и провести опрос аудитории перед запуском.
