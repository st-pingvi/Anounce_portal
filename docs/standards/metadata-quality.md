# Metadata Quality

## Обязательные показатели качества

Каждый сценарий оценивается по шкале от 1 до 5.

- `methodical_readiness`
- `host_clarity`
- `replicability`
- `district_adaptability`
- `engagement`
- `resource_accessibility`
- `practice_validation`
- `topic_relevance`

## Дополнительные флаги

- `piloted`
- `scaled`
- `needs_rework`
- `local_adaptation_required`
- `flagship_recommended`

## Итоговая оценка

`quality_score_total` — среднее арифметическое по 8 критериям, округленное до десятых.
